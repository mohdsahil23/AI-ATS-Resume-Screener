# 📄 AI ATS Resume Screener

## Overview

AI ATS Resume Screener is a Streamlit-based application that helps recruiters screen multiple resumes against a Job Description (JD). The application calculates ATS scores, ranks candidates, stores data in a database, and generates PDF reports.

---

## Features

- 📂 Multiple Resume Upload
- 📄 Job Description Upload
- 🎯 ATS Score Calculation
- 🏆 Candidate Ranking
- 👤 Candidate Profile
- 💾 SQLite Database
- 📊 Dashboard
- 📄 PDF Report Export
- 📊 Excel Export
- 🤖 AI Resume Analysis (Currently disabled due to Gemini API quota)

---
